# Accu Machserv Website

Production website for Accu Machserv — precision mechanical engineering services.

Live at: https://accumachserv.com

## Tech Stack
- Pure HTML/CSS/JS (single-file site)
- Hosted on Netlify with auto-deploy from GitHub
- Custom domain via GoDaddy
- WhatsApp integration for customer inquiries

## Local Development
Open index.html in any browser to preview.

## Deployment
Pushes to main branch auto-deploy to production via Netlify CI/CD.
